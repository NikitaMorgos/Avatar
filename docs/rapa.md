# RAPA / GTD — Концепт Avatar

5 главных баз. Archive — статус/флаг внутри баз.

## 1. Projects

Поля: Name, Area (→ Areas), Status (Idea/Active/On Hold/Done/Archived), Horizon (Week/Month/Quarter/Year), Impact, Effort, Start Date, Due Date, Next Review, Daily Focus Counter, Related Resources, Related Raw, Daily Logs.

## 2. Areas

Поля: Name (Business, Family, WCS, Ironman…), Type (Personal/Work/Creative), Goal (описание ответственности), Projects, Resources.

## 3. Resources

Поля: Name, Type (Template/Prompt/Script/Article/Note/Video/Course), Domain, Linked Projects/Areas, Source, Status (Active/Deprecated/Archived).

Сюда кладутся шаблоны и промпты из блока 25–40 и 40–50 (60‑минутный протокол).

## 4. Raw (Inbox)

Поля: Title, Content, Source (Email/Plaud/Chat/Manual/System), Created At, RAPA Stage (Raw/Assigned/Processed), GTD Type (Action/Idea/Reference/Trash), Assigned Project, Assigned Area, Converted Resource.

Всё от ассистента (Plaud, email, чаты) → Raw. Assign → Project/Area, RAPA Stage = Assigned.

## 5. Daily Log

60‑минутный протокол. Поля: Date, Day Of Week, Domain (Code/Agents, Product/Startup, Creative, Free), Focus Project, Focus Area, Focus Question, Success Criteria, What We Did, New Resources Created, Raw Captured, Automation Steps, Wins, Fails, Next Experiment Idea.

Секции: 0–5 вопросы фокуса, 5–25 диалог, 25–40 правка+шаблон, 40–50 автоматизация, 50–60 рефлексия.

## Представления

- **Dashboard Today** — Daily Log = today, Projects (Active), Raw (Stage=Raw), Resources (новые).
- **RAPA Radar** — Raw, Projects, Resources (7 дней), Daily Log (7 записей).

## Goals

Goals на год (Area + описание). API: `/api/rapa/goals?year=2026`.
