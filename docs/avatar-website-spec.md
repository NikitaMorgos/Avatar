# Сайт Avatar — спецификация

## Общая идея

Личный сайт‑витрина системы Avatar: описание концепта, области жизни, ссылки на модули и инструменты.

## Варианты реализации

### Вариант A: Статичный лендинг

- HTML + CSS + JS
- GitHub Pages / Netlify / Vercel
- Разделы: Концепт, Области жизни, Ссылки (Collect‑бот, канал и т.п.)

### Вариант B: Минимальный веб‑сервер

- Python (FastAPI/Flask) или Node.js
- Шаблоны для страниц
- Возможность позже встроить CMS (панель Collect) как подраздел

### Вариант C: Всё в одном

- Сайт Avatar + панель CMS Collect в одном приложении
- Главная — концепт, `/collect` — управление контентом

## Содержание (черновик)

- **Главная** — что такое Avatar, флоу (цель → спринт → задачи)
- **Области жизни** — список доменов (Business, Family, Sport и т.д.)
- **Модули** — Collect, ссылки на бота и канал, позже — другие
- **Ссылки** — TG‑бот, канал, контакты

## Папка проекта

- `web/` или `site/` — код сайта
- Либо отдельный репозиторий для статичного сайта (GitHub Pages)
