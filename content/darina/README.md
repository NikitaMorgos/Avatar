# Контент для книги Дарины

Сюда складывай все исходные материалы для книги.

## Структура

```
content/darina/
├── audio/       ← голосовые заметки и интервью (.m4a, .mp3, .wav)
├── photos/      ← фотографии по годам (называй: 2013_описание.jpg)
└── drawings/    ← рисунки Дарины (называй: 2015_описание.jpg)
```

## Как добавить контент

1. **Аудио-заметки** → положи в `audio/`, затем скажи Cursor:  
   «транскрибируй audio/название.m4a и добавь в книгу»

2. **Фотографии** → положи в `photos/`, затем через редактор  
   [darina.html](https://nikitamorgos.github.io/Avatar/darina.html)  
   загрузи напрямую в нужную запись

3. **Рисунки** → аналогично в `drawings/`
