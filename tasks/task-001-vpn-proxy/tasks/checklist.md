# Подзадачи — task-001-vpn-proxy

## Пользователь делает
- [ ] Создать VPS на Timeweb Cloud (Германия, Ubuntu 22.04)
- [ ] Получить IP-адрес VPS
- [ ] Сгенерировать SSH-ключ: `ssh-keygen -t ed25519 -f "$HOME\.ssh\timeweb_vpn" -N ""`
- [ ] Добавить публичный ключ в Timeweb → SSH-ключи
- [ ] Проверить вход: `ssh -i "$HOME\.ssh\timeweb_vpn" root@VPS_IP`

## Cursor делает (после получения IP)
- [ ] Создать `deploy/start-tunnel.ps1`
- [ ] Создать `deploy/start-avatar.ps1`
- [ ] Вставить IP в скрипты
- [ ] Обновить `.env` (раскомментировать PROXY)
- [ ] Установить зависимость: `pip install aiohttp-socks`

## Проверка
- [ ] SSH-туннель держится >5 минут без разрыва
- [ ] Бот стартует с логом `Using proxy: 127.0.0.1:1080`
- [ ] Отправить фото боту → получить ответ → фото появилось в БД
- [ ] Бот постит в канал по расписанию (20:00)
